Technical Artist

For the Caillou specials I built reusable systems and simulated elements. Some examples are- an infinitely-scaling ocean in redshift, with procedural elements to add interaction along the shoreline, wharf, and any docked boats. A Houdini fire simulation was exported as a re-usable cache, added to sets, and shaded in Redshift. A flickering candle setup was built as a reusable Nuke tool, to be added to shots later by compositors. Moving boat interactions were done with a combination of procedurally generated waves from Houdini and Nuke particles for the white water. A template for the underwater scenes was created in Nuke using a combination of colour corrects, depth passes, light rays and particles. Some research was done on using plates rendered from Houdini for snow interactions.
